## Project Overview
This project demonstrates how SQL can be used to design, populate,
and analyze a supply chain and inventory management system.

The database tracks suppliers, products, inventory levels, and orders
to support operational and analytical decision-making.

## Objectives
- Monitor current inventory levels
- Identify products below reorder level
- Track pending and delivered orders
- Calculate inventory value
- Analyze supplier-product relationships

## Tools & Technologies
- SQL (MySQL / PostgreSQL / SQL Server compatible)
- GitHub (Web-based version control)
- CSV datasets

## Database Tables
- Suppliers
- Products
- Inventory
- Orders

## Key Skills Demonstrated
- SQL table creation with constraints
- Data insertion and querying
- Joins, aggregations, and filtering
- Business-focused analysis
